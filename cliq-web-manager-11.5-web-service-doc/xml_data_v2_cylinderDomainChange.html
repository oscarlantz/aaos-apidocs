cylinderDomainChange Data Type

Represents a cylinder or cylinder group transfer event from one domain to another

Properties
name data type type namespace min/max occurs description

affectedGroup cylinderGroup element 0/1 The cylinder group that was moved
newDomainName string element 1/1 The new domain name
oldDomainName string element 1/1 The old domain name
updateTime dateTime element 1/1 When the change was made, ISO 8601 format for example 2014-09-05T16:15:30+02:00
updatedBy administrator element 0/1 Who made the changes
