normalAuditTrailEntry Data Type

NormalAuditTrailEntry is an audit trail entry that is the result of interacting with another CLIQ element in the same MKS.

Note that fields such as name, cylinder planned location etc are set at the time when the audit trail is imported to CLIQ Web Manager. Since some of these fields can be changed in CLIQ Web Manager, this data may have changed from the time the audit trail entry was created in the element till it was imported into CLIQ Web Manager.
