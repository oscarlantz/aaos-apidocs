keyDetails Data Type

Represents a detailed representation of a CLIQ or mechanical key, including static and dynamic info.

Namespace
http://cliq.shared.assaabloy.com/ws/data/v2/
Schema
data_v2.xsd
Properties
name data type type namespace min/max occurs description
bluetoothSupport bluetoothSupport element dataExtra_r7_0_v2 0/1 Indicates if key does support bluetooth communication.
deleted boolean element 1/1 Set to true to mark a key as deleted. A deleted key is still in the system for reference, but can not be used.
domainName string element dataExtra_v2 0/1 Key domain.
handinDate string element 0/1 Handin date.

Format: [YYYY]-[MM]-[DD]T[hh]:[mm]+01:00, e.g. 2011-06-16T18:25+01:00

handoutDate string element 0/1 Handout date.

Format: [YYYY]-[MM]-[DD]T[hh]:[mm]+01:00, e.g. 2011-06-16T18:25+01:00

identity string element 1/1 Key Identity.

Min length: 1
Max length: 36

keyHolderIdentity string element 0/1 Unique identification of the key holder. Will be null if key is not handed out.

Min length: 1
Max length: 256

lastRemoteUpdate string element 0/1 Timestamp when the key was last updated in a remote PD.
marking string element 1/1 Key marking.
name string element 0/1 Key name.

Max length: 100

operationalStatus operationalStatus element 1/1 Describes the key's current operational status.
pendingUpdates pendingUpdates element 1/1 Represents the current pending updates for this key.
pinValidationIntervalInMinutes int element dataExtra_r7_0_v2 0/1 PIN Validation interval in minutes.

The maximum value of PIN Validation interval is 45 days (64800 in minutes). 0 means PIN Validation is disabled for this key.

remoteEnabled boolean element 1/1 Indicates if the key is remote enabled.
secondMarking string element 0/1 Second marking.

type elementType element 1/1 CLIQ key type.
validity validity element 0/1 Represents the validity of this key.
whereAt whereAt element 1/1 Describes where the key currently is.
