cylinderSearchArguments Data Type

Represents arguments used for searching for cylinders. If an individual argument is left empty, it will match any value. If all arguments are left empty, any cylinder will be matched.

Namespace
http://cliq.shared.assaabloy.com/ws/data/v2/
Schema
data_v2.xsd
Properties
name data type type namespace min/max occurs description
domain string element 0/1 The name of the cylinder's domain.
group string element 0/1 The cylinder group name of the plugs in the cylinder.
location string element 0/1 The location of the cylinder.
marking string element 0/1 The marking of the cylinder.
model string element 0/1 The model name of the cylinder.
name string element 0/1 The name of the cylinder.
operationalStatus operationalStatus element 0/unbounded The operational status of the cylinders to search for. The default is both operational cylinders and broken cylinders.
secondMarking string element 0/1 The second marking of the cylinder.
secondName string element 0/1 The second name of the cylinder.
tags tags element 0/1 The set of tags applied to the cylinder. Cylinders will be matched only if they have all the specified tags.
type elementType element 0/unbounded The type of cylinder to search for. The default is both electronic cylinders and mechanical cylinders.
whereAt installationStatus element 0/unbounded The installation status of the cylinders to search for. The default is both in-stock cylinders and installed cylinders.
