accessProfileSearchArguments Data Type

Represents a set of search arguments for access profiles.

Namespace
http://cliq.shared.assaabloy.com/ws/data/v2/
Schema

Properties
name data type type namespace min/max occurs description
description string element 0/1 The string used for matching access profile descriptions. The description string can use asterisks (*) as wildcards.
domain string element 0/1 The string used for matching the access profile domain name. The domain name string can use asterisks (*) as wildcards.
identity string element 0/1 Access profile identity. Used for exact matching.
name string element 0/1 The string used for matching access profile names. The name string can use asterisks (*) as wildcards.
tags tags element 0/1 The tags used for matching access profile tags. Access profiles will be matched only if they have all the specified tags.
