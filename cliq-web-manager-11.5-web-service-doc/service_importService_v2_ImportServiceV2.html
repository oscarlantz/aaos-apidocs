ImportServiceV2 Service

Service for importing data into CLIQ Web Manager. Typically used to import data stored in other systems, eg. importing person data from a local AD into Web Manager.

Namespace
http://cliq.shared.assaabloy.com/ws/import/v2/
WSDL
importService_v2.wsdl
Path
/ImportServiceV2

savePersons Operation

Saves the given list of new person(s), all basic attributes of persons can be saved.

Any ASCII control chars in person fields eg in firstName will be removed.

Note:

  • if type is not defined, persons are defined as employees by default
  • if domain is not defined then person is assigned to the initial domain configured in system settings

Input Parameters
name type description
persons personDetails Persons to save. Id must be unique and must not exist.
Return Value
type description
batchResult Result after completed operation. If error occurs then batch result contains a list of failures grouped by persons.
Faults
name type conditions
SystemException SystemException Thrown if an unrecoverable error occurs. Error log should be available on server.

updatePersons Operation

Updates the given list of persons in CWM, all basic attributes of persons can be updated.

If the person exists, the existing person instance will be overwritten with new information. If the person does not exists, a new person will be created in CWM.

Note:

  • a failure will be added to the result if trying to delete a person who has a key
  • a failure will be added to the result if trying to update a person imported from LDAP
  • if domain is not defined then person is re-assigned to the initial domain configured in system settings

Input Parameters
name type description
persons personDetails Persons to update.
Return Value
type description
batchResult Result after completed operation. If error occurs then batch result contains a list of failures grouped by persons.

name type conditions
SystemException SystemException Thrown if an unrecoverable error occurs. Error log should be available on server.
