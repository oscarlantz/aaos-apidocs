cylinderUpdate Data Type

Represents a change of information for one cylinder. If a field is filled in (even with an empty string), it will replace the current value of that field in the cylinder. If a field is omitted, the current value for that field is kept.

Namespace
http://cliq.shared.assaabloy.com/ws/data/v2/
Schema
data_v2.xsd
Properties
name data type type namespace min/max occurs description
customFields customFields element dataExtra_r6_1_v2 0/1 Represents the set of updated custom fields to be applied for this cylinder. Each custom field will be replaced by a custom field with the corresponding identity. Cylinder custom fields which are not present in the set will remain unchanged.
externalLinks string element 0/1 A string of comma-separated external links related to this cylinder. These external links will replace the current external links on the cylinder.
identity string element 1/1 The identity of the cylinder to be affected by the update.

Min length: 1
Max length: 36

installationStatus installationStatus element 0/1 The new installations status, e.g. installed or in stock.
lengthA string element 0/1 The new length of cylinder plug side A.
lengthB string element 0/1 The new length of cylinder plug side B.
name string element 0/1 The new cylinder name.

Max length: 100

plannedLocation string element 0/1 The new planned location of the cylinder. Note: this field is called location in the CLIQ Web Manager application.

Max length: 255

secondName string element 0/1 The new cylinder secondary name.

Max length: 255

tags tags element 0/1 Represents the set of tags applied to this cylinder. These tags will replace the current tags on the cylinder.
